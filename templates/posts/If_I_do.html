If (I do)

Ifは「もし」という意味です。whenと同じで必然的に２つの節が必要です。

If　＋　節(ifパート)　＋　, (then)　＋　節(メインパート)
節(メインパート)　＋　if　＋　節(ifパート)

どの時制を使うかで当然意味は変わってきますが、本来の時制の意味と若干違うので注意が必要です。thenは入れても入れなくても大丈夫です。

ifパート - 現在形、 メインパート - 現在形/未来形/命令形

これが基本的な形になります。

例

If I get an email from Fred, I will let you know.
I will let you know if I get an email from Fred.

もしFredからメールが来たら知らせるよ。

Whenをifに変えた以外はwhen の時に紹介した文と全く同じです。この文は、Fredからメールが来るかどうか分からないということを暗示しています。

If you want to start a business, don't wait.

もし起業したいなら、待つな。

If you can’t explain it simply, you don’t understand it well enough.

簡潔に説明出来ないのは十分に理解していないからだ。 - Albert Einstein

ifパート - 過去形、メインパート - would

ifパートが過去形になると、「もし仮に起こったら」という意味になります。メインパートにはwouldを使います。過去形を使っているからといって過去の事を話しているわけではありません。

例

I don't have money. If I had money, I would buy a house.

私はお金が無い。もしお金があったら私は家を買うだろう。

I am not American. If I were American, I would not have to learn English.

私はアメリカ人ではない。もし私がアメリカ人なら英語を学ばなくていいだろう。

Ifを使う時はbe動詞にwereを使うことが出来ます。普段通りwasを使うことも出来ます。

What would you do if you won the lottery?

もし宝くじに当たったらどうしますか？

ifパート - 過去完了形、メインパート - would (not) have　＋　過去分詞

would haveはwould'veに省略出来ます。

これは過去の事を表現する時に使われます。

例

I did not have money. If I had had money, I would have bought a house.

私はお金がなかった。もしお金があったら私は家を買っただろう。

これは先ほどの例と違い、過去の事を話しています。

I was not busy. If I'd been busy, then I wouldn't have been able to help you.

私は忙しくなかった。もし忙しかったら、あなたの事を手伝う事が出来なかった。
